# Checkhouse

Мониторинг доступности для сервисов. Публичные и приватные статус страницы для отображения истории проверок мониторов.


<Image src="/works/checkhouse/home.png" width={1024} height={768} alt="homepage" />

В приложении можно начать с демо аккаунта для него нужен только никнейм. В течении 7 дней, если не привязаться почту аккаунт будет удален.

## Мониторы

В данный момент есть 3 вида мониторов: HTTP, TCP, DNS.

Также к HTTP и DNS мониторам можно добавить проверки, например, проверка будет успешной при наличии в ответе определенных значений.
Для HTTP можно проверять заголовки и статус код ответа, в DNS проверять значения записей.


## Статус страница

Статус страницы нужны для отображения истории мониторинга, коммуникации инцидентов. По умолчанию к странице можно выбрать `*-status.checkhouse.app` поддомен,
если у вас есть свой домен, можно подключить его, например, `status.example.com`
